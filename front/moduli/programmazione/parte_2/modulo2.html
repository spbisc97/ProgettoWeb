Cos'è la programmazione?

Un altro tipo in python è il Booleano, può avere due valori: True(vero) o False(falso). possono essere creati comparando due valori, oppure usando l'operatore logico (==). Un'altra comparazione logica è (!=). In python si possono fare anche le comparazioni matematiche di base (<),(>(<=)(>=),il loro valore di ritorno sarà un Booleano. Per far eseguire il codice a certe condizioni si può usare l'istruzione "if" se l'espressione che si controlla con if avrà valore "True" allora sarà eseguita, altrimenti no. D'altro canto, come quasi sempre succede, si ha la necessita che il codice continui anche se certe istruzioni non sono state rispettate. Per fare questo si usa l'istruzione "else". È possibile concatenare le istruzioni if ed else, per fare più controlli, ma Python ci offre una terza istruzione per fare ciò "elif". Per rendere più complicate le condizioni, è possibile usare gli operatori logici.

Come già detto l'istruzione if viene eseguita una volta se la condizione verificata è True. il ciclo "while" esegue un blocco di istruzioni finché la condizione esaminata dal while non diventa True. Il problema di questa condizione è che è facile ritrovarsi in un ciclo infinito, per questo bisogna stare attenti che la condizione possa diventare true. in alternativa si può mettere un contatore ed usare l'istruzione break all'interno del while per fare in modo che il ciclo si blocchi dopo un numero predefinito di tentativi. Un altro ciclo è il "for" che esegue un blocco di codice un determinato numero di volte, questo lo si può fare grazie all'istruzione in range.

Leliste sono un tipo di oggetto in Python, sono usate per memorizzare una lista di elementi indicizzata,una lista si crea con le parentesi quadre([]), lasciandole vuote avremo una lista vuola, per riempirle basta inserire, numeri o stringhe dentro la lista, e dividerle con delle virgole. Tipicamente la lista contiene un solo tipo tra i due, ma può contenerli entrambi. Per sapere quanto è lunga una lista c'è la funzione len. Gli elementi in una lista possono essere riassegnati. Una lista può essere molticata o divisa nello stesso modo delle liste. È possibile fare controlli nelle liste, controllare se un elemeno è o non è presente. Si utilizzano gli operatori in per controllare se è presente not il contrario. per aggiungere elementi ad una lista già creata si può usare append o insert, per rimuoverle remove. di seguito aggrego una tabella con le funzioni più usate sulle liste in Python. ed un esempio.
